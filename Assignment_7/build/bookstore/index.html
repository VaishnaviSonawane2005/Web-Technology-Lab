Assignment 7 - Servlet + MySQL

Digital Bookstore Inventory

This project demonstrates how a Java Servlet connects to MySQL, retrieves data from the ebookshop table, and performs complete inventory operations through the browser.

The application now supports viewing all books, searching by title or author, adding new books, editing records, deleting rows, and showing basic inventory statistics.

Open Inventory Dashboard View Low Stock Books

Technology Stack

Built for a Tomcat deployment using Jakarta Servlet API, JDBC, and MySQL Connector/J.

  • Java Servlet controller mapped to /books
  • JDBC-based MySQL connection
  • Prepared statements for CRUD operations
  • Responsive UI rendered directly by servlet output

Operations Demonstrated

  • Select: display all book details from MySQL
  • Insert: add new book records from the form
  • Update: edit existing book information
  • Delete: remove unwanted records

Database Table

  • book_id - integer primary key
  • book_title - title of the book
  • book_author - author name
  • book_price - price in rupees
  • quantity - number of copies available

Before Running

  • Execute sql/create_database.sql in MySQL
  • Place MySQL Connector/J JAR in Tomcat lib or WEB-INF/lib
  • Update DB values in web.xml if your credentials differ
