BOOKSTORE SERVLET APPLICATION - QUICK START

1. Create the MySQL database and table:
   mysql -u root -p < sql/create_database.sql

2. Make sure MySQL Connector/J is available:
   - Preferred: place mysql-connector-j-x.x.x.jar in CATALINA_HOME\lib
   - Or place it in WebContent\WEB-INF\lib

3. Update database credentials if needed:
   - Edit WebContent\WEB-INF\web.xml
   - Change dbUrl, dbUser, and dbPassword context-param values

4. Build the WAR:
   - Windows: build.bat
   - Linux/Mac: ./build.sh

5. Deploy:
   - Copy build\bookstore.war to CATALINA_HOME\webapps\

6. Start Tomcat and open:
   http://localhost:8080/bookstore/

FUNCTIONALITY INCLUDED
- Display all books from MySQL
- Search by title or author
- Filter low-stock books
- Add a new book
- Edit an existing book
- Delete a book
- Show inventory statistics
